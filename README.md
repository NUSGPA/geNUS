# geNUS (formerly NUSGPA)

geNUS is a beautifully designed, feature-rich academic planner and "Wrapped" generator exclusively built for National University of Singapore (NUS) students. It helps you track your academic progression, strategize your S/U options, and visualize your university journey with stunning, highly personalized shareable reports.

## ✨ Key Features

1. **Dashboard Manager**: Track modules, visualize your grade distribution, and monitor your cumulative GPA with modern, interactive metrics and donut charts.
2. **S/U Scenario Sandbox**: An advanced S/U calculator. Toggle S/Us on current or past modules to instantly see their numerical and visual impact on your GPA. Plan your academic safety net with confidence.
3. **geNUS Wrapped**: Discover your unique "Academic Archetype" generated by AI, read your academic fun facts (grade streaks, A-grade rates, and best semesters), and generate a stunning 480px, fully formatted Story Card ready to be shared on social media.

## 🛠️ Technical Architecture & Implementation

geNUS is built entirely with Python using the **Streamlit** framework seamlessly blended with custom CSS and client-side JavaScript.

### 🎨 Frontend & UI/UX
While Streamlit is traditionally suited for internal data apps, geNUS pushes the framework's boundaries to create a production-level Web App feel:
- **Custom CSS Magic**: Extensive CSS injections replace default Streamlit blockiness with smooth, mesh-gradient elements, responsive grid layouts, and custom typography (Google's "Outfit" font).
- **Client-Side Export**: The "Share Your Story" card export bypasses server-side image rendering. Instead, it injects an HTML block loaded with `html2canvas.js`, allowing the user's browser to paint the DOM element onto an HTML5 Canvas and trigger a direct `image/png` download locally on the user's device.

### 🧠 Generative AI & RAG Implementation 
The crown jewel of the Wrapped feature is the **Academic Archetype Generator**, which employs a **Retrieval-Augmented Generation (RAG)** approach backed by Google's **Gemini 2.5 Flash** model.

#### How the RAG Pipeline Works (`wrapped_engine.py`):
1. **Module Prefix Extraction**: The engine scans the user's transcript and extracts the 2-to-4 letter prefix from every module taken (e.g., `CS1010S` -> `CS`, `MA1521` -> `MA`). It calculates the frequency of each prefix to understand the user's "weighting" or core focus.
2. **Knowledge Retrieval (`filtered_raw.csv`)**: A lightweight, pre-processed static database containing NUS module codes and their full descriptive titles. The app cross-references the user's extracted prefixes against this database.
3. **Context Assembly**: The engine builds a rich text context block. Instead of just telling the AI "The user took CS 3 times", the RAG pipeline retrieves the true data and provides structural context: *"Prefix CS (taken 3 times): Programming Methodology, Data Structures and Algorithms, Software Engineering..."*
4. **LLM Prompting**: A carefully tuned prompt alongside the retrieved context is sent to the Gemini model. The model is instructed to act as a "student profiler", weighting the heavily taken modules to determine the core identity, and using the less frequent modules to add flavor to a 1-2 sentence description.
5. **Fallback Mechanisms**: If the Gemini API hits a rate limit (HTTP 429) or quota restriction, the engine features a gracefully degrading fallback mechanism. It strips the heavy descriptive RAG context and passes only the raw prefixes, maintaining the feature's availability without crashing the app.

## 🚀 Setup & Installation

To run geNUS locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/NUSGPA/geNUS.git
   cd geNUS
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Configure API Keys:
   Create a folder named `.streamlit` in the root directory and add a `secrets.toml` file:
   ```toml
   GEMINI_API_KEY = "your-google-gemini-api-key"
   ```
5. Run the app:
   ```bash
   streamlit run app.py
   ```
